# GBCS

Guangzhou Biobank Cohort Study.

## Repository Contents

- `GBCS/README.md`: merge workflow notes and latest run summary
- `GBCS/merge_plan.md`: step-by-step merge plan and QC rules
- `GBCS/merge_gbcs_main.R`: R script to build the merged cohort dataset (`gbcs_main.dta`)

## Data Handling

- Raw datasets and merged database files are not committed to GitHub
- The merged output (`gbcs_main.dta`) is generated locally (for example under `/Users/linxu/Documents/GBCS/`)
